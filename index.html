CSV Epoch Time Converter

Upload a CSV file containing a column named "time" with epoch timestamps (like 1744674946890266400 - nanoseconds assumed).

The script will convert the "time" column to the format YYYY-MM-DD HH:MM:SS.mmm and allow you to download the modified file.

3. Download Fixed CSV

Notes:

  • The script looks for a header column named exactly "time" (case-insensitive).
  • It assumes the epoch time is in nanoseconds
  • This script processes the file entirely in your browser. No data is sent to any server.
  • Limitation: This uses basic CSV parsing (splitting by lines and commas). It might not work correctly with complex CSVs containing commas within quoted fields or escaped quotes. For robust parsing, a dedicated library would be needed.
  • Large files might take some time or cause browser performance issues.
